## 📌 About the Course
The BLoC pattern separates presentation/UI from business logic in Flutter applications. This playlist covers state management concepts using the `flutter_bloc` package in Arabic, focusing on clean architecture, reactivity, and maintainable state handling.

- **Instructor:** Omar Ahmed
- **Playlist Link:** [Flutter BLoC - بالعربي](https://www.youtube.com/playlist?list=PLwWuxCLlF_ufA0GYYjlx_R4smekKH_AuB)

---

## 🛠 Prerequisites

Before starting with BLoC, make sure you have:
* **Flutter SDK:** `>=3.0.0`
* **Dart SDK:** `>=2.17.0`
* Basic knowledge of Flutter Widgets, Streams, and Reactive Programming (`RxDart` concepts).

---

## 📚 Key Modules Covered

1. **Introduction to State Management**
   * Why BLoC?
   * Difference between Ephemeral (UI) state and App State.
2. **Cubit vs. BLoC**
   * Simplified state management using **Cubit** (functions to emit states).
   * Event-driven state management using **BLoC** (Events in, States out).
3. **Core BLoC Widgets**
   * `BlocProvider` & `MultiBlocProvider`
   * `BlocBuilder`
   * `BlocListener` & `MultiBlocListener`
   * `BlocConsumer`
   * `RepositoryProvider`
4. **Advanced Concepts & Best Practices**
   * Handling side-effects (Navigation, SnackBars).
   * State immutability & Equatable integration.
   * Asynchronous state handling (Loading, Success, Error states).
   * Practical real-world project structure.
